# Spotify-Display

A simple project that displays currently playing Spotify track information.

## Features
- Shows current song title
- Displays artist name
- Updates in real-time
- Button features like pause/continue, fast forward, rewind
  
## Setup
- Clone the repository
- Install dependencies
- Add your Spotify API credentials
- Run the project

## Usage
- Start the application and view your currently playing track.
